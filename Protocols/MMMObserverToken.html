MMMObserverToken

Objective-C

@protocol MMMObserverToken <NSObject>

Swift

protocol MMMObserverToken : NSObjectProtocol

Sort of a cookie which can be returned by different addObserver methods. It allows to remove the observer explicitely using #remove method or implicitely when the token is deallocated.
