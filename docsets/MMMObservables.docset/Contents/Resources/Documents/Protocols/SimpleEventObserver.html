SimpleEventObserver

Protocol for objects that want to be notified in a “classic” delegate-like manner when an event they subscribed to via SimpleEventObservable.addObserver() is triggered.

  • simpleEventDidTrigger(_:)

    The exact instance of SimpleEventObservable that was used when subscribing to the event is passed so the observer can distinguish between different events it has subscribed to.
